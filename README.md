# Site-Laura

J'ai réalisé ce site web dans le cadre d'un projet académique.
Il présente l'unniversitaire brittanique Andrew Hoskins, son parcours ainsi que ses réalisations. Ses travaux s'articulent autour de l’idée de mémoire connectée : il s’intéresse à la façon dont les technologies numériques changent notre rapport au passé, influencent ce dont nous nous souvenons et transforment nos mémoires collectives.
